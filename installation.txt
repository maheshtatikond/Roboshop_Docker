
Install below packages to install docker
sudo dnf -y install dnf-plugins-core

 sudo dnf config-manager --add-repo https://download.docker.com/linux/rhel/docker-ce.repo
 sudo dnf install docker-ce docker-ce-cli containerd.io docker-buildx-plugin docker-compose-plugin

 sudo systemctl start docker
 sudo systemctl enable docker
 sudo systemctl status docker 

# Need to add, user to the docker group. to get docker command access.
sudo usermod -aG docker ec2-user

Home location of docker Docker/var/lib/docker 
While ceating the ec2 instance I have allocated 50GB storage 
df -hT = to check the file system

we allocated 2 GB to   /dev/mapper/RootVG-varVol    xfs       2.0G  583M  1.4G  30% /var

now we need to claim 50 GB storage 
To resize disk:
sudo growpart /dev/nvme0n1 4
sudo lvextend -L +30G /dev/RootVG/varVol
sudo xfs_growfs /var
